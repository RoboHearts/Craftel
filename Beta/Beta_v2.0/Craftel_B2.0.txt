◆About this Project
Craftel is a robot platform made from paper that is powered by the wind.
By using everyday items such as skewers and rubber bands, and printing the published patterns, anyone can start building robots.
website: http://craftel.org

◆Update
1.1からの変更点
- 大きな変更点 -
・竹串
・輪ゴム（小）
・セロテープ
これらを使わずに作れるようになりました。
これに伴って、挿し込み構造や軸、新しい部品などが加わりました。
 
- 小さな変更点 -
材料など
・135kgから180kgのより厚い用紙に変更しました。
・キットに使用していたプラスチック袋（小）が不要になりました。

顔・目
・まぶたとまぶた台を一体化しました。
・まぶたを挿し込めるように顔の下にまぶた挿し込み口を設けました。
・目が2種類から選べるようになりました。
・目の瞳の大きさを大きくするなど目のデザインを変更しました。
・まぶたの角度をより水平にして、従来の眠たい感じの顔からよりシャキッとした感じの顔になりました。
・顔の奥行きを小さくしました。
・顔と脚との連結が簡単になりました。

脚
・リンクC-6の強度を高めました。
・プーリーの両サイドの板を少し大きくすることで輪ゴムが落ちにくくなりました。
・電池や石などのおもりを包む構造をシャーシに設けました。

その他
・風車を15°傾けることで用紙の左右の余白が大きくなりました。
・型紙のライセンスをCC-BY-SA　から CC-BY-NC-SAに変更しました。
・全体的により鮮やかな色になりました。
・部品名が見やすくなりました。
・トンボからの余白をわかりやすくしました(Position line layer)。
・各色を印刷しやすくするためにひとつのファイルにまとめました。
・教室などで使用するための.pdfマニュアルを作りました（現在日本語版のみ）
・解説に使用するための３Ｄモデルを作り公開しました。


◆Note
・型紙データについて
Adobe Illustrator (2017)にて製作しました。
レイヤー
GC... : カッティングプロッタ―に使用するトンボです。カッティングの際の位置ずれを防止します。
PartsName : 部品名のテキストとその背景です。
Others : その他のテキストやロゴです。
Cut(first) :　切れ込み線　(ミシン目なし、カット速度 2cm/s)　細かい切り抜き部分や先に切り抜く必要がある部分などに使用しています。
Cut : 　切り込み線　(ミシン目あり、カット速度 8cm/s)　通常の切れ込みを入れます。
Fold : 折り目の線　ハーフカットで切ります。
Texture : 部品などに使用する塗部分です。
Position line : 範囲を表示しています。
Back ground : 背景とグリッドです。

・使用しているカッティングプロッター
Graphtec CE6000-40 Plus

・キット製作に使用しているプリンター
Canon LBP841C (Laser printer)

・DIYについて
プリンタと用紙(A4,180kg)、輪ゴム(#30)、カッター、テープがあればDIY可能です。
ただ、細かい構造などのカットが難しいため、可能であれば最初はキットを購入して頂くか、
カッティングマシンを使用してカットすることをおすすめします。

・データの使用について
ライセンスの範囲内においてご家庭や教室、ワークショップなどで自由に使って頂いて大丈夫です。
ただ、商用でのデータの使用や複製したキットのイベントなどでの多め(50個以上)の配布などをお考えの場合は必ずお問合せください。
また、作り方がわからない場合やキットの購入などについてもお気軽にお問い合わせください。
メールでのお問い合わせ: yoshida@shellmo.org
